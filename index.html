Theme Settings

Colors :

Customize the colors of various elements on the theme, including the main header, homepage, product pages, blog, and footer.

  • Home header background:

    Used for changing the background of the top main header.

  • Home header icons color:

    Used for changing the colors of the top main header icons.

  • Home header navigation text color:

    Used for changing the text color of the top main header navigation.

  • Separator line color:

    Used for changing the header's separation bottom line color if it's enabled (you can enable it from the header).

  • Sticky header background:

    Used for changing the background color of the sticky header if it's anabled (YOU CAN ENABLE FROM THE HEADER SECTION).

  • Sticky header icons color:

    Used for changing the colors of the the sticky header icons.

  • Sticky header text color:

    Used for changing the text color of the sticky header navigation.

  • Drawer Background:

    Used for changing background color of the drawers like the menu and cart on mobile.

  • Drawer Text:

    Used for changing the text color of the drawers.

  • Borders color:

    Used for changing the borders color inside the drawers.

  • Search Background:

    Used for changing background color of the drawers like the search container.

  • Megamenu Background:

    Used for changing background color of the Megamenu (if it's anabled).

  • Megamenu Primary Text Color:

    Used for changing Main text color of the Megamenu.

  • Search Background:

    Used for changing Main text color of the Megamenu.

  • Primary Heading Color:

    Used for changing Primary heading color of the website.

  • Secondary Heading Color:

    Used for changing Secondary heading color of the website.

  • Primary Text Color:

    Used for changing Primary Text color of the website.

  • Secondary Text Color:

    Used for changing Secondary Text color of the website.

  • Sub Text Color:

    Used for changing third (used for subtitles) Text color of the website.

  • Primary Background:

    Used for changing Primary Background color this would be also the background of the website.

  • Secondary Background:

    Used for changing Secondary Background color of the sections you can choose this color on any section you want.

  • gradient Background:

    Used for changing Gradient background color.

  • Button background:

    Used for changing Primary/Secondary Button background color.

  • Button background on Hover:

    Used for changing Primary/Secondary Button background color when the mouse is hovering over it.

  • Button label:

    Used for changing the text color of the Primary/Secondary Button.

  • Button label on Hover :

    Used for changing the text color of the Primary/Secondary Button when the mouse is hovering over it.

  • Paragraph Link Color :

    Used for changing the text color of the paragraph links.

  • Buy now Button background :

    Used for changing buy now Button background color.

  • Buy now Button label:

    Used for changing buy now Button text color.

  • Primay Overlay color:

    Used for changing the Primary Image Overlay color (used in Image banner,collection images).

  • Secondary Overlay color:

    Used for changing the Secondary Image Overlay color (used in Image banner,collection images).

  • Current Price :

    Used for changing the Current Price text color.

  • Previous Price :

    Used for changing the Previous Price text color if it's exist.

  • Amount Saved :

    Used for changing the Saved price text color.

  • Capitalize:

  • banner's heading:

    used the change the size of the headings in (image banner, slideshow, email...)

  • title's Heading size:

    used the change the size of the headings in other sections like (featured collection, tistmonials...)

  • Headings Spacing:

    used the change the spacing between heading letters

  • body text font:

    used to change the type of the font of the website text

  • text size:

    used the change the size of the website text

  • body text Spacing:

    used the change the spacing between text letters

  • Navigation Heading font:

    used to apply the headings font to the header navigation menu

  • Navigation bold font:

    use the bold font in the header navigation menu

Buttons:

the typography that you can change in the theme settings.

  • Primary Button:

    used to change style of the Primary button:

    • - Button outline style

      used to add border to the buttton and a transparent background

    • - Button radius

      used to add border radius to the buttton

    • - Button anable hover effect

      used to add style to the button when the mouse is over it

    • - Button hover effect

      used to select the hover effect animation from: 'top/left/bottom/right'

  • Secondary Button:

    used to change style of the Secondary button:

    • - Button outline style

      used to add border to the buttton and a transparent background

    • - Button radius

      used to add border radius to the buttton

    • - Button anable hover effect

      used to add style to the button when the mouse is over it

    • - Button hover effect

      used to select the hover effect animation from: 'top/left/bottom/right'

Layout:

You can change the width of the container of the website from the "Page width" and "Articles width".

  • Page width:

    used to change the width of the sections on the page

  • Articles width:

    used to change the width of the Contact and the Blog article pages

Badges:

You can change the colors of the badges on the product card like "Sale", "Slod out", "New".

  • Sale badge:

    used to change background and text colors for the 'Sale' badge

  • Sold out badge:

    used to change background and text colors for the 'Sold Out' badge

  • New badge:

    used to change background and text colors for the 'New' badge

  • Header Cart badge:

    used to change background color of the 'Cart' badge on the header (when user add products to the cart a little circle displays with the cart icon)

Social media:

You can add your social media links so you can use them on your website footer/header.

Favicon :

You can add your favicon to your website, the favicon is the icon of your website that displays on the tab.

Filters:

You can change the colors of the Selected filter and Price range slider on the collection page.

  • Price range slider :

    used to change background color of the slider circle and progress on the collection page

  • Selected Filter :

    used to change background and text colors of the selected filter on the collection page

Product Card:

Used to change the style of the product card.

  • Product Title Color :

    used to change the color of the product card title

  • Title font type :

    used to change the type of the product title card font: Heading/Body

  • Add Product Card Radius :

    used to add radius to the product card

  • Show second image on hover :

    used to display the secondary image of the product when hovering over the product card

  • Show Vendor :

    used to add the Vendor

  • Product Image Style :

    used to change the product card image ratio

  • Image Radius :

    used to add radius to the product card image

  • Text alignment :

    used to change the product card text alignment Left/Center/Right

  • Enable Color Swatches :

    used to display other product variant on the product card

  • Selected Variant Color :

    used to change the color of the selected variant

  • Hide Saved Price :

    used to add/hide the amount saved for "On Sale" products

Variant Pills:

Used to change the colors of the variant pills in the product description.

  • Pills Background :

    used to change default background color of the product variants

  • Pills Background :

    used to change default text color of the product variants

  • Pills Background :

    used to change default border color of the product variants

  • Pills Background :

    used to change the background color of the selected product variants

  • Pills Background :

    used to change the text color of the selected product variants

  • Pills Background :

    used to change the border color of the selected product variants

Slideshow :

The same as the image banner but with multiple images.

Section settings

  • Image height :

    Sets how tall you want the slides to be, adjust depending on your images..

  • Allow the image animation :

    Used to add zoom-in animation the image.

  • Add arrows :

    Used to add prev/next buttons.

  • Anable Autoplay :

    Controls whether or not the sides changes the images automatically.

  • Change slides every :

    Controls how often the slides change.

Block settings

  • Slide :

    Each slide is an image banner you can add title,text,link and mobile/Desktop images.

Video Banner :

Section settings

  • Section Full width :

    By anable in this the video will take the full width of the website.

  • Video Source (from local files) :

    Use this field to link to a file from your shopify admin Check how to upload files .

  • Hide Icons Even when hovering :

    Shows pause/play/sound buttons (Depending on the type of video and how you use this section, you may want to give the customers the ability to go back and forth and pause the video, or to just use it decoratively), You can also change the controls color.

  • Autoplay :

    Controls if the video will play by itself without user interaction (if you have controls hidden, you need to enable autoplay, otherwise the customers won't be able to play the vide)..

  • Loop :

    Controls if the video will play again if the video ends.
